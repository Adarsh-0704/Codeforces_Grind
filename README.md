Codeforces Problem Finder
This is a responsive web application built to help competitive programmers find the right type and rating of problems to solve to get a rank up.

API Used:
https://codeforces.com/api/problemset.problems

Planned Features
1) Display a list of practice problems on the screen.
2) Let users filter problems by topic and rating.
3) Order the ratings of the problems in increasing and decreasing order.
4) Let users search for the name of the problem they are looking for.

Tech Stack to be used
HTML
CSS
Javascript
